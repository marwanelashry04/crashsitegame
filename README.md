# CrashSite

A 2D space shooter game built in Unity where you pilot a spaceship against waves of alien invaders across three increasingly difficult levels.

## Gameplay

- Control a spaceship and shoot lasers at descending alien ships
- Survive three levels of escalating difficulty
- Face a **boss battle** in the final level

## Built With

- Unity
- C#

## How to Play

1. Clone the repository
2. Open the project in Unity
3. Press Play in the Unity Editor or build for your platform
4. Use arrow keys / WASD to move and spacebar to shoot

## Levels

| Level | Description |
|-------|-------------|
| 1 | introductory wave of alien ships |
| 2 | faster and more aggressive enemies |
| 3 | boss battle with unique attack patterns |
